
LoginActivity displays a login form, allows user to enter username and password
- user can click on the Connect to eXo button to create start a new Activity called HomeActivity
- this HomeActivity displays a Hello message to user
- the Intent will carry username to the HomeActivity
- when the user enters username and password, LoginActivity will check those against a HashMap
if the username or password is incorrect, LoginActivity will display a "username or password incorrect" box
